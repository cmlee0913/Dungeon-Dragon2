# Dungeon-Dragon2
전투 후 얻는 DropItem의 활용이 중심인 기믹을 적용시킨

스테이지를 클리어해 핵심 아이템을 습득하는 필드 액션게임.

각각 다른 기믹이 적용된 4개의 스테이지를 클리어하고

획득한 아이템으로 보스전의 난이도를 완화시킬 수 있습니다.
